# Notes app backend

## Description
Backend made in NestJS for a notes app that lets you CRUD notes in .md format. 
Created Notes are saved locally in the /notes folder so they can be easily
used for export or other purposes.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
